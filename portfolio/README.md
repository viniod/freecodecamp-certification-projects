# 💼 Personal Portfolio

Portfólio pessoal desenvolvido para apresentar projetos e formas de contato.

## 🚀 Tecnologias
- HTML5
- CSS3
- Flexbox
- CSS Grid

## 📱 Funcionalidades
- Seção de apresentação
- Grid de projetos
- Links para redes sociais
- Layout responsivo

## 🎯 Objetivo
Criar um portfólio profissional para exibir projetos e iniciar na área de desenvolvimento web.
