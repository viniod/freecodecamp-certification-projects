# 🎧 Product Landing Page

Landing page de um produto desenvolvida como parte da certificação de Responsive Web Design do freeCodeCamp.

## 🚀 Tecnologias
- HTML5
- CSS3
- Flexbox

## 📱 Funcionalidades
- Layout responsivo
- Menu de navegação fixo
- Seção de planos em formato de cards
- Vídeo incorporado

## 🎯 Objetivo
Praticar a criação de páginas de produto com foco em responsividade e layout moderno.
